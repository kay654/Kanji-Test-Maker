# 漢字テスト作成

小学生向けの漢字書き取りテストを作成する静的Webアプリです。

## 機能

- 問題文、問題の漢字、よみがなを入力してA4縦書きプリントをプレビュー
- ランダム生成で学年別の問題を作成
- 回答文字の表示・非表示切り替え
- CSVテキストによるインポート、CSVファイルのダウンロード
- 最大48問、A4最大2ページ

## 使い方

`index.html` をブラウザで開くか、静的ファイルとしてホスティングしてください。

ランダム生成を使う場合は、`data/kanji-data.js` を同じ配置で配信してください。

## データについて

同梱の漢字問題データは、複数の第三者データソースを元に生成されています。
詳しくは [THIRD_PARTY_NOTICES.md](THIRD_PARTY_NOTICES.md) を確認してください。

生成データは教材作成を補助するためのものです。公開・配布・授業利用の前には内容を確認してください。
