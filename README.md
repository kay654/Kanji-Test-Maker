# 漢字テスト作成

小学生向けの漢字書き取りテストを作成する静的Webアプリです。

## 機能

- 問題文、問題の漢字、よみがなを入力してA4縦書きプリントをプレビュー
- 手入力の問題では、必要な場合だけ全文の読みを追加して他の漢字にも読み仮名を表示
- ランダム生成で学年別の問題を作成
- 漢字1字から利用できる例文を検索して問題へ追加
- ランダム生成した問題文では、出題漢字以外の漢字にも読み仮名を表示
- 回答文字の表示・非表示切り替え
- CSVテキストによるインポート、CSVファイルのダウンロード
- 最大48問、A4最大2ページ

## 使い方

`index.html` をブラウザで開くか、静的ファイルとしてホスティングしてください。

ランダム生成を使う場合は、`data/kanji-writing-questions.js` を同じ配置で配信してください。

## データについて

同梱の漢字問題データは `kanji-writing-question-dataset` v2.0.0 で、文全体の読みと表示範囲を含みます。複数の第三者データソースを元に生成されています。
詳しくは [THIRD_PARTY_NOTICES.md](THIRD_PARTY_NOTICES.md) を確認してください。

生成データは教材作成を補助するためのものです。公開・配布・授業利用の前には内容を確認してください。
